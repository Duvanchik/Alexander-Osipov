Тег picture

Тег <picture> позволяет выбрать наиболее рациональное по своему качеству (разрешению) изображение для устройства на котором оно будет отображаться, а также позволяет решить проблему художественного оформления на web-странице: например когда широкое изображение при сужении области видимости (viewport) уменьшается и становится плохочитаемым, то его при определенной ширине viewport необходимо заменить на заранее подготовленное, но более узкое, которое будет смотреться на узком экране гораздо лучше. Также <picture> может использоваться для отображения в браузере картинок подходящего формата благодаря возможности задания в одном тэге нескольких атрибутов <source>.
Ознакомиться с работой этого тега можно на данной страничке зайдя в инструменты разработчика и воспользовавшись инструментом device tool bar. При просмотре страницы в Google Chrome следует также отключить в инструментах разработчика отключить кэш на вкладке Network или вручную обновлять сраницу с очисткой кэша при изменении ширины viewport для более наглядной работы тега. Это связано с тем, что Chrome, когда загрузит более качественное изображение при, например, расширении viewport не станет возвращать изображение с меньшим разрешением при сужении области видимости в изначальное состояние.
